Engineering Capstone Project

Flight Control System Optimisation for a Cessna 182

Role: Flight Dynamics & Simulation Engineer

This engineering capstone investigated the optimisation of a Cessna 182 flight control system using MATLAB and Simulink. Working within a multidisciplinary engineering team, the project integrated nonlinear flight dynamics, simulation, manual Genetic Algorithm optimisation and atmospheric disturbance modelling to evaluate aircraft performance under varying operating conditions.

Project Information

Project Type Final-Year Engineering Capstone
Role Flight Dynamics & Simulation Engineer
Aircraft Cessna 182
Outcome High Distinction

Engineering Challenge

Develop a simulation-driven workflow capable of evaluating aircraft controller performance while maintaining stable longitudinal flight characteristics under realistic atmospheric disturbances.

My Responsibilities

  • Developed the nonlinear three-degree-of-freedom longitudinal flight dynamics model.
  • Implemented the aircraft simulation environment in MATLAB and Simulink.
  • Developed and refined custom MATLAB S-functions.
  • Performed aircraft trim analysis for steady-state operating conditions.
  • Integrated a manually implemented Genetic Algorithm workflow with the flight simulator.
  • Implemented Dryden atmospheric gust modelling.
  • Analysed aircraft stability and dynamic response.
  • Supported project integration, verification and validation within the engineering team.

Technical Approach

Flight Dynamics Model

Developed a nonlinear longitudinal model representing aircraft airspeed, angle of attack, pitch attitude, pitch rate, altitude and range.

MATLAB and Simulink

Used MATLAB for mathematical implementation, parameter evaluation and data analysis, while Simulink provided the integrated dynamic simulation environment.

Aircraft Trim

Established steady-state flight conditions before optimisation and disturbance testing to support stable and repeatable simulation runs.

Manual Genetic Algorithm

Integrated a manually implemented Genetic Algorithm workflow to evaluate, compare and refine candidate parameter sets through iterative simulation.

Dryden Gust Model

Applied atmospheric turbulence inputs to evaluate aircraft response beyond ideal still-air operating conditions.

Verification and Validation

Checked aircraft state behaviour, trim conditions and simulation outputs through progressive subsystem and integrated-system testing.
